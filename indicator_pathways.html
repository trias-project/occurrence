3.1 Find categories

We divide the species in categories. Each kingdom, except Animalia, is a category. If animal species are present, they are separated in two categories: vertebrates (phylum Chordata) and invertebrates (phylum other than Chordata, defined as Not Chordata).

3.2 Pathways associated with alien species introductions

Function get_table_pathways() from trias library is used for creating tables based on checklist data. Species category and number of species returned as example per each pathway modality are provided.

Number of species returned in table as examples of the group:

n_species <- 5

We limit to pathways of taxa introduced from 2000.

start_year <- 2000

Create tables:

tables <- map(categories$category, ~ 
                get_table_pathways(data, 
                                   category = ., 
                                   from = start_year,
                                   n_species = n_species))
names(tables) <- categories$category

3.3 Pathways per category

Intentionality of introduction is mapped by cell color background: white for intentional introduction (release), light orange for unintentional introduction.

3.4 Pathways related to policy relevant species

It is important to focus our attention also to a intercategory group of species which is of particular policy importance: the List of Invasive Alien Species of Union concern (see more info about here).

We import this list of species:

eu_concern_file <- here("data","input", "eu_concern_species.tsv")
eu_concern_species <- read_tsv(eu_concern_file)

Check which species from this subgroup are not present in data:

eu_concern_species %>% filter(!backbone_taxonKey %in% data$nubKey)

Categories present in this subgroup:

As done before, we limit to pathways of taxa introduced from 2000.

Create tables:

tables_eu_concern <- map(categories_eu_concern$category, ~ 
  get_table_pathways(data %>%
                       filter(nubKey %in% eu_concern_species$backbone_taxonKey), 
                     category = ., 
                     from = start_year,
                     n_species = n_species))
names(tables_eu_concern) <- categories_eu_concern$category
